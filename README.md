## 1. 담당 구현 기능

- 개발중

<br/>

## 2. 개발 환경 세팅

```
git clone https://github.com/Avocado-StudyGroup/zero-waste.git
cd zero-waste
pnpm install
npm dev
```

<br/>

## 3. 기술 스택

- vite, react, typescript, tailwindcss, axios, react-router-dom

<br/>

## 4. 커밋 컨벤션

```
feat : 새로운 기능 추가
fix : 버그 수정
docs : 문서 수정
refact : 코드 리팩토링
style : 코드 의미에 영향을 주지 않는 변경사항
chore : 빌드 부분 혹은 패키지 매니저 수정사항
conf : config 수정
rename : 파일명, 폴더명 수정 또는 이동
remove : 파일 삭제
```
